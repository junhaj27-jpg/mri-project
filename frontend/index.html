Brain MRI 3D Visualization & Volume Measurement Web Service

공개 저장소에는 코드, 공개 가능한 데모 이미지, mock longitudinal data만 포함합니다. 실제 DICOM/NIfTI 원본과 3D 분석 산출물은 비공개 로컬 환경에서만 사용합니다.

MRI 시점 보기 3D 뷰어 확인

본 시스템은 연구용 분석 보조 프로토타입이며 의료진의 진단을 대체하지 않습니다.

데이터 흐름

Demo Mode는 Kaggle 2D demo와 mock tracking만 보여줍니다. Private Analysis Mode는 로컬 private DICOM/NIfTI volume을 입력으로 받아 3D mesh와 부피 결과를 생성합니다.

Demo Mode

공개 저장소용 데모

Kaggle 2D demo 이미지와 mock longitudinal data만 사용합니다. 이 모드는 화면 시연과 UI 흐름 확인용이며 실제 3D 부피 계산 입력으로 사용하지 않습니다.

Private Analysis Mode

비공개 로컬 분석

실제 DICOM/NIfTI volume, segmentation mask, GLB mesh, volume result는 `data/private`, `data/dicom`, `data/nifti`, `outputs/private` 같은 로컬 폴더에서만 사용합니다.

FastAPI

정적 페이지, REST API, SQLite 기반 mock 분석 결과 조회를 담당합니다.

Private MRI Pipeline

pydicom, nibabel, NumPy, marching cubes 기반으로 로컬 private volume을 처리합니다.

Three.js Viewer

생성된 GLB 모델을 브라우저에서 회전, 확대, 축소하며 확인합니다.
