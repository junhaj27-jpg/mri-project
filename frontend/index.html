Brain MRI 3D Visualization & Volume Measurement Web Service

Brain MRI 기반 3D 병변 시각화 및 segmentation mask 기반 부피 계산 웹서비스입니다. 원본 DICOM/NIfTI 파일은 공개 저장소에 올리지 않고, 비식별 분석 결과만 조회합니다.

MRI 시점 보기 3D 모델 확인

본 시스템은 의료진 진단을 대체하지 않는 연구용/포트폴리오용 프로토타입입니다.

핵심 흐름

DICOM/CD 정리 → NIfTI 변환 → 2D slice 확인 → mask overlay → voxel 기반 cm³ 부피 계산 → 3D mesh 생성 → PC/모바일 웹 확인
