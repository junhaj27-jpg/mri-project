mri-project
대시보드 MRI 목록 2D 뷰어 부피 3D 뷰어
Private Analysis Mode
실제 3D 분석은 private NIfTI/DICOM volume과 segmentation mask 입력을 기준으로 실행합니다.

Demo Mode에서는 샘플 구조 모델만 표시합니다. 실제 GLB, mask, volume 결과 파일은 GitHub에 업로드하지 않습니다.

3D 구조 모델 뷰어

Private Analysis Mode에서 생성한 구조별 GLB를 대뇌, 소뇌, 뇌간, 해마, 종양 단위로 켜고 끌 수 있습니다. 화면을 스크롤하면 3D 모델이 함께 회전합니다.

Demo Mode

샘플 구조 모델만 표시합니다. Kaggle 2D demo는 실제 3D 부피 계산에 사용하지 않습니다.

Private Analysis Mode

로컬 private NIfTI/DICOM volume에서 mask와 GLB를 생성한 뒤 ignored media/output 경로에서 불러옵니다.

Demo Mode의 샘플 구조를 표시합니다. 마우스 드래그 또는 스크롤로 3D 모델을 움직일 수 있습니다.

원본 MRI를 직접 공개하지 않고, private mask에서 생성한 결과만 로컬에서 확인합니다.
