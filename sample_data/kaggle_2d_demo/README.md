# Kaggle 2D Demo Data

Place public Kaggle-style JPG/PNG MRI slices here. Public labels are separated by anatomy and task:

- `brain_mri/tumor/`
- `lumbar_mri/normal/`

Use `masks/` for public 2D reference mask/overlay examples. These masks are for demo and fine-tuning preparation only. Do not use them for private 3D volume measurement.
